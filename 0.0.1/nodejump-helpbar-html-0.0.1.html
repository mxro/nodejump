Linking

The prime purpose of nodejump is not to write one web page but rather to create a number of interconnected pages at a rapid pace.

There are numerous ways to create new pages and links between existing pages.

Insert Link to New Page

Just click the button to insert a link to a new document at the current cursor position.

Insert Link to Existing Page

In order to insert a link to an existing page, you must first get the address of the page you want to link to.

The address of each page is displayed on top of the document preview. You can right click this link and copy the link address to obtain a link.

You can then insert the obtained link using either Markdown formatting or plain HTML:

[my link](http://your-link-here.com)

<a href="http://your-link-here.com">my link</a>

Navigation

To jump from one document to another, just click on the links shown in the document preview.

Otherwise, you can use the browser navigation buttons like on any other webpage. So clicking the back button will bring you back to the last documents you have visited.

You can also use your browsers history, favourites and bookmarks as convenient ways to access your documents.

Lastly, if you are logged in, all your documents will be saved in your user node. Open the settings to see your user node and all documents and data stored under your node.

Publish to the Web

If you are logged in with your appjangle account, your documents will by default be created as private documents (as long as they are not created as children of an already published document).

You can click on the share link on top of the document preview and click on to make your document available for the public web.

If you are not logged in, all your documents will be created as public by default.

Share by Link

You can generate unique links for your documents, which will allow to edit and/or view the document and all of its children.

Just click on the share link on top of the document preview and select the respective function in the share dialog.
